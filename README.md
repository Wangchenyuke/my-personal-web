# 我的个人网站

***

## 项目介绍

本项目是展示我的个人信息的网页，包含家乡文化、家乡特产、家乡景点等介绍

## 框架

- Spring Boot 2.7.x
- Spring MVC
- MyBatis + MyBatis Plus 数据访问
- Spring Boot 调试工具和项目处理器
- 前端使用JSP

## 业务功能

- 个人信息的展示
- 实现家乡文化、家乡特产、家乡景点的增删改查。
